### How To Use:
```bash
[+] CDN Url : https://cdn.jsdelivr.net/gh/dbstorage/cdn/(FILE Name).(Extension)

[+] Example Usage :
<script type="text/javascript" src="https://cdn.jsdelivr.net/gh/dbstorage/cdn/js/jquery.min.js"></script>

```
